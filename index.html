Hitman

The professional RSS enclosure downloader. Supports OPML and various download methods: system wget, system curl, and urlgrabber. Hitman is written in Python with a Terminal interface, and is entirely open-source.

  • Alias feeds with keywords.
  • Supports system wget, system curl, urlgrabber (pycurl based).
  • Cross-platform folder support.
  • Progress meter on UNIX OSes (and Windows if Termios support is enabled).
  • Download notifications using Growl or OSD-notify.
